
# 💜 GlamBook - Beauty Parlour Management App

A modern, responsive web application for managing beauty parlour operations.

## ✨ Features

- 📊 **Dashboard** - Real-time overview of bookings, revenue & stats
- 📅 **Appointments** - Book, manage & track appointments
- 💇 **Services** - Add/edit/delete services with pricing
- 👥 **Customers** - Customer database with visit history
- 👩‍💼 **Staff** - Team management with availability status
- 💰 **Billing** - Generate & print invoices
- 📈 **Reports** - Business analytics & popular services
- ⚙️ **Settings** - Configure parlour details

## 🚀 Live Demo

Visit: `https://YOUR-USERNAME.github.io/beauty-parlour-app/`

## 📦 Setup

1. Fork/Clone this repository
2. Open `index.html` in your browser
3. That's it! No build tools needed.

## 🌐 Deploy on GitHub Pages

1. Go to your repo **Settings** → **Pages**
2. Select **Branch: main** → **/ (root)**
3. Click **Save**
4. Your app will be live in 1-2 minutes!

## 🛠 Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Poppins)
- No frameworks needed!

## 📱 Responsive

Works perfectly on Desktop, Tablet & Mobile devices.
